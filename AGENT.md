# Project Notes

## Overview
This is a rapidocr test project - prototype of OCR using PP-OCRv5 on ONNX Runtime.

## Environment
- **OS**: Windows
- **Python**: Use `py` command (Windows Python launcher)
- **Virtual Environment**: `.venv` (already exists in project root)

## Setup

### Activate Virtual Environment
```bash
.venv\Scripts\activate
```

### Run Python Scripts
```bash
py script.py
```

## Model
- **Model**: korean_PP-OCRv5_mobile_rec
- **Location**: `korean_PP-OCRv5_mobile_rec/`
- **Framework**: ONNX Runtime
